tweet2toot
===

A simple script that transport tweets from twitter to Mastodon. Based on the Twitter RSS feed powered by [RSSHub](https://rsshub.app).

```
pip3 install -r requirements.txt
cp conf.sample.ini conf.ini
vi conf.ini
python3 run.py
```
